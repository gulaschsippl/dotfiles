## Install

	git clone https://github.com/pangratz/dotfiles.git ~/.dotfiles
	cd ~/.dotfiles
	script/install
	script/bootsrap

## Set sane OSX defaults

	cd ~/.dotfiles
	osx/set-defaults.sh

## Apply changes

	cd ~/.dotfiles
	script/bootstrap
